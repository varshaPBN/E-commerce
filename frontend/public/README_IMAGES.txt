2. PRODUCT IMAGES (for Store Setup page)

      File name: tshirt.jpg or tshirt.png
      Description: White t-shirt with three cartoon cats graphic
      
   c) BEDROOM
      File name: bedroom.jpg or bedroom.png
      Description: Bedroom scene with botanical art prints above bed
      
   d) CAP
      File name: cap.jpg or cap.png
      Description: Light beige baseball cap with pink bow

===========================================
HOW TO ADD IMAGES
===========================================

1. Copy your image files to this folder: frontend/public/
2. Make sure file names match exactly (case-sensitive)
3. Supported formats: .jpg, .jpeg, .png
4. Refresh your browser after adding files
